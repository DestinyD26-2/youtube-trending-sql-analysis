# YouTube Trending SQL Analysis

## 📊 Overview
This project analyzes YouTube trending video data to understand what drives user engagement. The analysis focuses on metrics such as views, likes, dislikes, and comment counts.

## 🎯 Objective
To identify patterns in high-performing videos and determine how engagement metrics influence trending content.

## 🛠️ Tools Used
- SQL
- Dataset: YouTube Trending Videos (U.S.)

## 🔍 Key Questions
- Which videos receive the most likes, dislikes, and comments?
- How does engagement change across rankings?
- What patterns exist among top-performing videos?

## 📈 Key Insights
- Videos with high dislikes often also have high comment counts, suggesting controversy drives engagement.
- The top-ranked videos have significantly higher comment counts compared to lower-ranked ones.
- Engagement drops sharply after the top 10, then becomes more consistent.
- Similar patterns are observed across likes, dislikes, and views.

## 📂 Files
- `queries.sql` → All SQL queries used in the analysis
- `insights.md` → Written analysis and conclusions
